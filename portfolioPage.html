Portfolio Projects

Number Guesser

Number Guesser was another project focused on javascript, this is a basic game in which the player chose a number, using the up and down arrows, between 1 and 10. Then once the program runs it will generate a number for the player's computer opponent and a random number that the players have to get close to. The final part is the program compares the players number and the computers number to judge which is closer to the initial number, and returns whether the player has won or lost. This is another project that was part of the Codecademy course and the HTML and CSS was already provided with myself adding in the Javascript functionality. A link to the project on my github page is: Number Guesser.
